Como começar?

Cole o link do seu perfil Steam na barra de busca acima e clique em BUSCAR.

FORMATO 1 — Link com nome personalizado https://steamcommunity.com/id/seuNome
FORMATO 2 — Link com ID numérico https://steamcommunity.com/profiles/76561198XXXXXXXX
FORMATO 3 — Só o número do Steam ID 76561198049642502

Onde encontro meu link do perfil Steam?

1
Abra o Steam (programa no PC ou pelo navegador)
2
Clique no seu nome no canto superior direito e vá em "Ver meu perfil"
3
Copie o link que aparece na barra de endereço do navegador.
Será algo como: https://steamcommunity.com/id/seuNome
4
Cole na barra de busca acima e clique em BUSCAR!
Importante: Seu perfil Steam precisa estar público para que os dados sejam carregados.
Para verificar: Steam → Perfil → Editar Perfil → Configurações de Privacidade → "Meu perfil: Público".

Outro jeito de encontrar seu ID:

Abra o Dota 2 → clique no seu perfil no canto superior esquerdo → o número que aparece abaixo do seu nome é o Friend ID. Cole ele na busca!
